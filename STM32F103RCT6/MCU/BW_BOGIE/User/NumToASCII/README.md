# Usage
This file converts a 32bits number to a ASCII string. <br>
For example:<br>
`0xFFFF FFFF`->`"0b1111 1111 1111 1111 1111 1111"`<br>

## Configuration
Configure your platforms in *num_to_ascii.h*.<br>
* Currently supported:
```
GCC_PLATFORMS (Default)
STM32_PLATFORMS
```
